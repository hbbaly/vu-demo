# cnode-vue

> 使用vue写cnode网站

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# lint
npm run lint

# fix
npm run fix

```
